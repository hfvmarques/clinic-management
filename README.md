# Descrição
## A API
Esta é uma RESTful API construída em _JavaScript_ e _Postgres_ para gerenciamento de uma clínica médica.

Os principais cenários de comportamento estão sendo cobertos por teste via `jest`.

## Como rodá-la
Para ter o ambiente de desenvolvimento, deverá instalar o gerenciador de pacotes [_yarn_](https://yarnpkg.com/) e o _nodejs_ na versão 18.1.0.

* Primeiramente clonar o repositório e rodar o comando `yarn` na raiz para instalar todas as dependências do projeto;
* O comando para rodar a API é `yarn start`;
* Para rodar a aplicação de forma que os arquivos salvos não necessitem matar e reiniciar a API é `yarn start:dev`;
* Para rodar os testes é `yarn test`;
* Para rodar a aplicação com `--watch` é `yarn test:live`;
